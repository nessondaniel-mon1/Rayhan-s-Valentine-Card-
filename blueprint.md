# Rayhan's Valentine Card

## Overview

This project is a simple, interactive Valentine's Day card for Rayhan. It's a single-page web application built with HTML, CSS, and JavaScript. The goal is to create a personalized and visually appealing digital card.

## Design and Features

*   **Theme:** Valentine's Day
*   **Typography:** A decorative font for headings and a clean, readable font for the main text.
*   **Color Scheme:** A palette of reds, pinks, and whites.
*   **Interactivity:** The card will have some interactive elements, like a button to reveal a message.
*   **Content:** The card will contain a personalized message for Rayhan.

## Project Structure

*   `index.html`: The main HTML file.
*   `style.css`: The CSS file for styling.
*   `main.js`: The JavaScript file for interactivity.
*   `blueprint.md`: This file, which documents the project.
*   `.gitignore`: A file to specify which files and directories to ignore in Git.
